Tutorial

Welcome to Random Tree, the algorithm that shows the beauty of fractal structures.
To generate a tree you can play with some parameters which influence in the process of generating the fractal.

You can define the number of iteration of the fractal (Iteration) and also set the coloring of the tree and background.

Here on the tutorial you won't find expecific explanation on how the algorithm works because even the creator doesn't understand how it does. The fractal art is a mystery and that is its beauty.

To learn about it you have to experiment yourself and if you find some beautifull trees you can download an image file on "Download".
